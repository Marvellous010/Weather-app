Een moderne weer applicatie die real-time weersinformatie toont voor elke stad ter wereld.

## Functies

- **Real-Time Weer** - Krijg actuele weergegevens voor elke stad
- **Temperatuur** - Bekijk de huidige temperatuur in Celsius
- **Vochtigheid** - Zie het vochtigheidspercentage
- **Windsnelheid** - Bekijk de windsnelheid in km/h
- **Weericonen** - Visuele weergave van het weer (zon, wolken, regen, sneeuw, mist)
- **Responsief Design** - Werkt op desktop en mobiel

## Installatie

1. Clone of download dit project
2. Open `Weer.html` in je browser
3. Zoek een stad en bekijk het weer!

## Gebruik

1. Typ een stadsnaam in de zoekbalk
2. Klik op de zoekknop
3. Bekijk de weersinformatie

## API

Deze app gebruikt de [OpenWeatherMap API](https://openweathermap.org/api) voor weergegevens.
